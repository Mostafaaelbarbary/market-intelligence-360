# Market Intelligence 360

An end-to-end data engineering project that ingests internal and external data, models it into a Customer 360 warehouse, validates data quality, orchestrates pipelines, and applies CI/CD engineering practices.

## Stack
- PostgreSQL
- Airbyte
- BigQuery
- dbt
- Airflow
- Great Expectations
- pytest
- GitHub Actions
- Superset or Looker Studio

## Project Goal
Build a modern ELT pipeline that combines internal customer data with external market signals into a Customer 360 gold layer.

## Planned Phases
1. Project foundation
2. Source systems setup
3. Airbyte ingestion
4. Warehouse setup
5. dbt modeling
6. Customer 360 gold model
7. Airflow orchestration
8. Data quality checks
9. Unit tests
10. CI/CD
11. BI dashboard
12. Documentation
